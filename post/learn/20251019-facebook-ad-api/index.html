Facebook 广告系列数据获取方法

2025-10-19
facebook
694 words 2 mins read

Contents

  • 创建应用
  • 图谱API
  • 参考

通过Facebook的广告管理工具和图谱API获取广告数据，包括获取用户商户资产、广告账户、广告系列、广告组、广告、广告报表数据。

获取广告系列数据，需要通过Facebook的Graph API获取数据。首先要创建一个Facebook开发者应用，并获取应用ID和应用密钥。

在开发者账号中创建应用后，需要将应用添加到Facebook开发者工具中，并获取应用权限。

并且要将账号添加进入相应的广告资产中，才能获取当前账户权限下的相应的数据。

创建应用

登录Facebook开发者平台，创建应用，如果还不是开发者账号，需要先申请开发者账号。

申请开发者账户和创建应用流程都很简单，流程如下图：

  1. 输入应用名称
  2. 选择用例，这里只是用来获取广告系列数据，后续只需要市场营销API权限，所以不选择用例
  3. 业务账号
  4. 要求
  5. 概览

按照要求填写即可创建成功，创建成功后可以访问应用工具，并获取应用权限。

这里是要用这个应用获取广告数据，所以需要获取应用权限，这里需要获取市场营销API权限，所以产品选择市场营销API。

然后在工具中选择生成口令及口令的权限，并生成口令，这里生成的口令有效期一般为2小时，是一个临时口令，所以需要保存好，后续可以使用这个口令加上应用的ID和密钥来获取长期口令，长期口令的有效期为60天。

图谱API

获取广告系列数据，需要通过Graph API获取数据。

Facebook 提供的 图谱API探索工具 非常实用，可以用来快速的查询API及其对应的参数。
