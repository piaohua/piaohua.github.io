学习如何在家看食材做菜

2025-04-18
cook
304 words 1 min read

Contents

How to Cook 程序员做饭指南

之前收藏过一个菜谱网站食用手册，你可以在网站上选择自己的食材，然后就可以帮你

搜索和筛选对应的食谱，这样在家就不用纠结做什么菜，对不会做菜的新手小白来说，特别友好，

而且每个食谱还有对应的B站视频教程，还提供菜谱搜索，对于选择困难患者，还可以点随机，

让程序来决定今天吃什么。

今天又发现一个有趣的菜谱网站How to Cook 程序员做饭指南，不仅包含各种种类的

菜谱，还包含做菜之前准备知识，以及各种做菜的专业知识，特别适合学习。

对于程序员来说，这个菜谱网站还可以自己部署起来，在docker中执行下面命令直接运行即可。

1
2

docker pull ghcr.io/anduin2017/how-to-cook:latest
docker run -d -p 5000:5000 ghcr.io/anduin2017/how-to-cook:latest

也可以下载源码程序员在家做饭方法指南直接用MkDocs在本地运行。
